# admin-system

> 该项目为一个使用[Ant Design Pro](https://pro.ant.design)创建的后台项目模板。

## 常用方法

1、安装依赖

```bash
# 在项目根目录下执行
npm install
```

2、本地开发

```bash
npm run start
```

3、正式打包

```bash
npm run build
```

4、检测代码风格

```bash
# 仅检查代码风格
npm run lint

# 检查代码风格的同时，自动修复部分可修复的问题
npm run lint:fix
```

5、跑测试用例

```bash
npm run test
```
